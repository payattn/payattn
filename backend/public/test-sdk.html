üìã What's Happening:

  • The SDK requests an ad from your extension via postMessage
  • Extension responds with ad creative (headline, body, CTA, destination)
  • SDK renders the ad in the slot below
  • SDK tracks impression using Intersection Observer (1+ second visible)
  • SDK reports impression to backend for settlement
  • Settlement triggers 3 Solana transactions (70/25/5 split)
