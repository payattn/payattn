‚ùì FAQ
Is my API key safe?
Yes. Your API key is stored in Chrome's local storage and never transmitted to PayAttn servers. Only the extension can access it.
What data goes to Venice AI?
Only the data you explicitly send for processing (profile info, ads to analyze). Nothing else is transmitted.
Can PayAttn access my data?
No. All processing happens locally in your browser. PayAttn servers never see your profiles, API keys, or processed data.
What if I uninstall the extension?
All data, including your API key, is automatically deleted. No traces remain.
